# 🧠 ML Algorithms Implementation

A hands-on collection of **Machine Learning algorithms implemented from scratch in Python**.  
The goal of this repository is to demystify ML by building algorithms step-by-step — covering math, logic, and visualization — without relying solely on high-level libraries.

---

## 📚 Algorithms Covered (Planned & Implemented)

- **Regression**
  - Linear Regression
  - Logistic Regression ✅
- **Classification**
  - K-Nearest Neighbors (KNN)
  - Support Vector Machines (SVM)
  - Naive Bayes
- **Decision-Based**
  - Decision Trees
  - Random Forests
- **Clustering**
  - K-Means
  - Hierarchical Clustering
- **Dimensionality Reduction**
  - Principal Component Analysis (PCA)
- **Neural Networks**
  - Perceptron
  - Multi-Layer Neural Networks (basic backpropagation)

Each algorithm will include:
- Mathematical intuition
- Python implementation
- Visualizations (decision boundaries, loss curves, etc.)
- Evaluation metrics
